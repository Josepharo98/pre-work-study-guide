HTML

  • HTML (Hyper Text Markup Language) is the structure of the content that goes inside the tags. Browsers provide default styles, but they aren't pretty.
  • HTML attributes are always defined in the beginning element, never the closing element.

CSS

  • CSS (Cascading Style Sheets) is used to style and layout web pages
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • CSS describes how elements should be rendered on screen, on paper, in speech, or on other med

Git

  • touch : Creates a files inside a directory
  • pwd : Displays the current working directory
  • cd : Navigates between directories in the path
  • ls : Displays the list of files and folders in the current diretory
  • mkdir : Creates a new directory
  • git status : Checks what branch we are currently on
  • git checkout -b branch-name : creates a new branch and switches to it

JavaScript

  • JavaScript is the tool that takes a static HTML/CSS webpage and makes it interactive
  • Variables are containers that store values. You start by delcaring a variable with the "let" keyword, followed bt the name of the variable. End it with a ";"
  • Arrays : This is a structure that allows you to store multiple values in a single reference. It's a single variable that is used to hold a group of data. Arrays are typically used to hold data that is related in some way.
  • String : This is a sequence of text known as a string. To signify that the value is a string, enclose it in single or double quote marks.
  • Number : This is a number. Numbers don't have quotes around them.
  • Boolean : This is a True/False value. The words true and false are special keywords that don't need quote marks.
  • Object : This can be anything. Everything in JavaScript is an object and can be stored in a variable. Keep this in mind as you learn.
