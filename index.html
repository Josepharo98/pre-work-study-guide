Prework Study Guide

HTML

  • HTML (Hyper Text Markup Language) is the structure of the content that goes inside the tags. Browsers provide default styles, but they aren't pretty.
  • HTML attributes are always defined in the beginning element, never the closing element.

CSS

  • CSS (Cascading Style Sheets) is used to style and layout web pages
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • CSS describes how elements should be rendered on screen, on paper, in speech, or on other med

Git

  • touch : Creates a files inside a directory
  • pwd : Displays the current working directory
  • cd : Navigates between directories in the path
  • ls : Displays the list of files and folders in the current diretory
  • mkdir : Creates a new directory
  • git status : Checks what branch we are currently on
  • git checkout -b branch-name : creates a new branch and switches to it
